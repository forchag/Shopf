{% trans "Selling and buying on Shopf" %}

{% blocktrans trimmed %}Follow these steps to publish new inventory, control pricing, and understand what the customer sees from discovery to delivery. The Shopf dashboard keeps everything in one place so you can move from setup to sales with confidence.{% endblocktrans %}

{% trans "Merchant workflow" %}

{% trans "Add a product and manage its offer" %}

{% blocktrans trimmed %}Store owners manage products from the dashboard area that is available after signing in with an administrator account.{% endblocktrans %}

  • 1.

    {% trans "Open the admin product list" %}

    {% blocktrans trimmed %}Navigate to Dashboard → Admin → Products → Show to review existing inventory, or go directly to the create form at /dashboard/admin/products/create/.{% endblocktrans %}

  • 2.

    {% trans "Describe the item" %}

    {% blocktrans trimmed %}Provide the product name, categories, detailed description, media gallery, and any tags customers should use when searching.{% endblocktrans %}

  • 3.

    {% trans "Set pricing and availability" %}

    {% blocktrans trimmed %}Define the base price, optional discount, SKU and stock quantity. You can revisit the edit screen to adjust pricing whenever campaigns change.{% endblocktrans %}

  • 4.

    {% trans "Publish and monitor" %}

    {% blocktrans trimmed %}Save the form to publish the product. Incoming orders appear under Dashboard → Admin → Orders where fulfillment, status updates, and refunds are managed.{% endblocktrans %}

{% trans "Need profile permissions?" %}

{% blocktrans trimmed %}Only staff members with admin privileges can access the product management tools. Add a teammate from the member settings if you need additional help.{% endblocktrans %}

{% trans "Pro tip" %}

{% trans "Reuse drafts" %}

{% blocktrans trimmed %}Save products as drafts while you source imagery or finalize pricing. Drafts stay hidden from customers until you publish.{% endblocktrans %}

{% trans "Buyer journey" %}

{% trans "What customers experience" %}

{% blocktrans trimmed %}Once a product is published it immediately becomes part of the storefront. The steps below describe what a shopper does to complete a purchase.{% endblocktrans %}

  1. {% trans "Discover products" %}

    {% url 'shop:list_grid' as shop_url %}

    {% blocktrans trimmed with shop_url=shop_url %}Customers browse the catalogue at {{ shop_url }}, filter by categories, or open a product detail page to review descriptions, price, and availability.{% endblocktrans %}

  2. {% trans "Add to cart" %}

    {% blocktrans trimmed %}From the product page they select quantity and press Add to cart. The mini cart (basket icon) updates instantly so they can keep shopping or proceed to checkout.{% endblocktrans %}

  3. {% trans "Checkout" %}

    {% url 'cart:cart-summery' as cart_url %} {% url 'order:checkout' as checkout_url %}

    {% blocktrans trimmed with cart_url=cart_url checkout_url=checkout_url %}At {{ cart_url }} shoppers review their basket, apply coupons, and continue to the secure checkout at {{ checkout_url }} where shipping and payment details are collected.{% endblocktrans %}

  4. {% trans "Order tracking" %}

    {% blocktrans trimmed %}After paying, customers land on the confirmation screen and can revisit Dashboard → Customer → Orders to see status updates whenever they sign in.{% endblocktrans %}

{% trans "Communication tips" %}

  • {% trans "Use tickets for support" %}

    {% blocktrans trimmed %}Customers can open tickets via the contact page. Respond quickly from the admin ticket panel to maintain trust.{% endblocktrans %}

  • {% trans "Keep inventory accurate" %}

    {% blocktrans trimmed %}Update stock levels after each shipment so out-of-stock products disappear from the storefront automatically.{% endblocktrans %}

  • {% trans "Automate invoices" %}

    {% blocktrans trimmed %}Download invoices from the order detail view and send them with shipping confirmations for a premium experience.{% endblocktrans %}
